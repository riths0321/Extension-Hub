How CSS Specificity Works

CSS specificity determines which rule wins when multiple rules target the same element. It's represented as a 4-part tuple: (a, b, c, d).

  • a — Inline styles (style="…")
  • b — ID selectors #id
  • c — Class / attribute / pseudo-class .cls, [attr], :hover
  • d — Element / pseudo-element div, ::before

Comparison goes left to right. The first position where they differ picks the winner. If all equal, source order decides.

#content .text p → (0,1,1,1)
body .container p → (0,0,1,2)
First wins — has an ID (b=1 vs b=0)
