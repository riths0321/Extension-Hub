How CSS Specificity Works

CSS specificity determines which rule wins when multiple rules match the same element. It's a 4-part tuple (a, b, c, d).

  • a — Inline styles  style="…"
  • b — ID selectors  #id
  • c — Class / attribute / pseudo-class  .cls, [attr], :hover
  • d — Element / pseudo-element  div, ::before

Comparison is left-to-right. First position where they differ picks the winner. Equal specificity → source order decides.

#content .text p (0,1,1,1)
body .container p (0,0,1,2)
