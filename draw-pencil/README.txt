Draw Pencil Pro
Version: 13.0

Advanced study-focused Chrome extension with drawing tools, reading mode and a blue Manrope UI system.

Core features
- Reading mode with clean article extraction
- Reading ruler for line-by-line focus
- Highlight capture from selected text
- Auto-generated flashcards from page selections
- Bookmark capture for revision points
- Focus timer with persisted progress
- Study summary sidebar inside reading mode
- Export study notes as JSON
- Drawing tools: pencil, highlighter, eraser, line, arrow, rectangle, ellipse, text, sticky notes and laser
- Grid overlay, pressure feel, dim mode, undo/redo, PNG save and clipboard copy

Design system
- Header color: #2563EB
- Button color: #2563EB
- Heading text: #111111
- Sub-heading text: #6B7280
- Background: #FFFFFF
- Font family: Manrope

Shortcuts
- Alt+D: Toggle the extension
- P/H/E/L/A/R/O/T/N/K: Drawing tools
- M: Toggle reading mode
- U: Toggle reading ruler
- G: Toggle grid
- Ctrl/Cmd+Z: Undo
- Ctrl/Cmd+Shift+Z or Ctrl/Cmd+Y: Redo
- Ctrl/Cmd+S: Save PNG
- Ctrl/Cmd+E: Export study notes
- [: Decrease brush size
- ]: Increase brush size
- Esc: Close reading mode or extension

How to use
1. Load the folder in Chrome using Developer Mode.
2. Click the extension icon or press Alt+D.
3. Use drawing tools on any page.
4. Select page text to save highlights, flashcards or bookmarks.
5. Open Reading Mode for a distraction-reduced study view.
6. Use Export Notes to download your study data.
