Privacy Policy

Last updated: April 8, 2026

README Generator is designed to work locally in your browser. The extension does not send your project descriptions, generated README content, or license selections to external servers.

What Data Is Processed

The extension processes the text you enter in the popup so it can generate a README preview and export file. This processing happens locally on your device.

What Data Is Stored

The extension stores a small set of preferences using Chrome storage:

  • Your last project description
  • Your selected template
  • Your selected license

What Data Is Not Collected

  • No analytics data
  • No advertising identifiers
  • No remote logging
  • No account information
  • No personal data transmitted to third parties

Permissions

The extension requests the `storage` permission only. This is used to save your local extension preferences.

Third-Party Services

The popup does not rely on third-party APIs, remote fonts, or external scripts.
