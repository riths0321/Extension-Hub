Overview

About This Tool

Quick Guide

UTM Link Builder helps you create clean campaign URLs with the right tracking parameters so you can measure traffic in analytics tools like GA4.

It also lets you save presets, reuse recent links, copy in multiple formats, and generate a QR code for fast sharing.

Getting Started

How To Use

1

Enter your base URL

Paste the page link you want people to visit, such as your landing page or product page.

2

Add required UTM values

Fill in Source, Medium, and Campaign. Add Term, Content, or UTM ID if you need deeper tracking.

3

Copy or share the result

Your tracked URL is generated automatically. Copy it as a plain URL, Markdown link, or HTML anchor.

4

Reuse with history and presets

Use History to reopen past links and Presets to save common source and medium combinations for faster work.
