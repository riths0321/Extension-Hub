# Random Password Generator Extension

A secure, local browser extension to generate strong random passwords using the **Indigo Night** theme from the UI Theme System.

## Features
- Customizable password length (8–32 characters)
- Options for uppercase, lowercase, numbers, symbols
- One-click copy to clipboard
- Completely local — no data sent online
- Themed with **Indigo Night** (security-focused)

## Installation
1. Download or clone this repo.
2. Open Chrome/Edge and go to `chrome://extensions/`
3. Enable **Developer mode**
4. Click **Load unpacked** and select this folder.

## Theme
This extension uses **Theme 3: Indigo Night** as per the design guide for security tools.

## Icons
Place your `icon16.png`, `icon48.png`, `icon128.png` in the root folder.
