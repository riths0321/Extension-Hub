# Microphone Checker Extension

Microphone Checker is a local Chrome extension to test microphone input in real time.

## Features
- Start/stop microphone test in popup
- Real-time audio level meter (dB)
- Peak and average level tracking
- Optional frequency response graph
- Diagnostics log with success/warning/error states
- Settings + help modal

## Privacy
- No external websites used for testing.
- No user data is sent to any server.
- All audio processing happens locally in popup via Web Audio API.

## Install
1. Open `chrome://extensions`
2. Enable Developer mode
3. Click `Load unpacked`
4. Select `microphone-checker`
