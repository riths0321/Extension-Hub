# 📄 Text to PDF Converter - PDFForge

A powerful Chrome extension that converts text to PDF instantly. **Offline-first**, **private**, and **feature-rich**.

## ✨ Features

- **Instant PDF Generation** - Convert text to PDF with one click
- **Completely Offline** - Works without internet connection
- **Privacy Focused** - No data leaves your browser
- **Smart Formatting** - Automatic text formatting options
- **Customizable** - Full control over fonts, margins, page size
- **Watermark Support** - Add custom watermarks to your PDFs
- **Page Numbers** - Automatic page numbering
- **Presets** - Quick style presets for different use cases
- **Settings Persistence** - All settings auto-save

## 🎨 Typography

### Primary Font: Manrope
- Modern geometric sans-serif font
- Included locally in the extension
- Fallback stack: Manrope → -apple-system → BlinkMacSystemFont → Segoe UI → Roboto → Helvetica Neue → Arial → sans-serif

### PDF Fonts Supported:
- **Sans-Serif**: Helvetica, Arial, Verdana, Tahoma
- **Serif**: Times New Roman, Georgia, Garamond, Baskerville
- **Monospace**: Courier, Courier New, Monaco, Consolas

## 🚀 Installation

### From Source (Developer Mode)
1. Download or clone this repository
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode" (top right)
4. Click "Load unpacked"
5. Select the extension folder
