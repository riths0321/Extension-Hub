🔁 ConvertAll – All-in-One Unit Converter
🧩 Description

ConvertAll is a fast, lightweight, all-in-one unit converter built right into your browser.
No more Googling “cm to inches” or “kg to lbs” — convert values instantly, even offline, directly from the toolbar.

Designed for everyday use with a clean, distraction-free interface.

🚀 Features

All-in-One Conversion

Length

Weight

Temperature

Volume

Time

Numbers

Instant Results – Converts as you type

Swap Units – Reverse conversion with one click

Offline Support – No internet required

Clean & Minimal UI – Fast and easy to use

🛠️ Tech Stack

HTML5 – User interface

JavaScript (Vanilla) – Conversion logic

CSS3 – Styling

Chrome Extension (Manifest V3) – Browser integration

⚙️ Installation (Developer Mode)

Clone or download this repository

Open chrome://extensions

Enable Developer mode (top-right)

Click Load unpacked

Select the convertall folder

🧠 How It Works

Select Category
Example: Length, Weight, Temperature

Enter Value
Example: 100

Convert Instantly
Uses predefined conversion factors

Example: cm → inches = value × 0.3937

All calculations run locally inside your browser.
